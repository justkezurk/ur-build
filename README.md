# ur.space

**Browser-based development lab** — Copy, paste, preview, all on your device.

## Current Features
- HTML + CSS, React/TSX, Python support
- Live preview (iframe)
- Pyodide Python execution
- Project saving (localStorage)
- Basic AI Assistant stub

## Next Steps (In Progress)
- Full BYOK (Bring Your Own Key) for any OpenAI-compatible API (Grok, Claude, Gemini, OpenAI, etc.)
- Encrypted local storage with user passphrase
- Multi-provider switching

## How to Use
Simply open `index.html` in any modern browser (Chrome recommended).

## Self-Hosting
This repo is set up for easy GitHub Pages deployment.

Built for privacy-first, zero-cost AI-assisted development.
